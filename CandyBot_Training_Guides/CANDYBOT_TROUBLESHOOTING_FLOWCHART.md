## ?? QUICK DIAGNOSTIC TABLE

| Symptom | Most Likely Cause | Quick Fix | Time |
|---------|------------------|-----------|------|
| "Connection Failed" | Expired/wrong token | Reset token in portal | 2min |
| "Invalid Channel ID" | Wrong ID format | Copy ID again from Discord | 1min |
| Empty user list | SERVER MEMBERS INTENT off | Enable + reset token | 3min |
| Messages not sending | Discord ? unchecked | Check the checkbox | 5sec |
| Messages not receiving | MESSAGE CONTENT INTENT off | Enable + reset token | 3min |
| "Guild Members error" | Intent enabled after token | Reset token | 2min |
| DMs not working | User blocked bot | Ask user to unblock | Vary |
| Bot shows offline | Not connected | Click CONNECT | 1min |
| Slow user list | Large server | Wait 30 seconds | 30s |
| App freezing | Multiple issues | Restart app | 30s |

## ?? DIAGNOSTIC COMMANDS

### Check Connection Status
```
Look for: "Connected to #channel-name" (GREEN)
If shows: "Not Connected" (GRAY)
Action: Click CONNECT button
```

### Check Discord Checkbox
```
Look for: ? Discord (CHECKED)
If shows: ? Discord (UNCHECKED)
Action: Click to check it
```

### Check Mode
```
Look for: ? Channel OR ? Direct Message
If wrong mode selected
Action: Click correct radio button
```

### Check User Selected (DM Mode)
```
Look for: "DM with [Username]" banner
If not visible in DM mode
Action: Click "SELECT USER" button
```

## ?? EMERGENCY PROCEDURES

### LEVEL 1: MINOR ISSUE (< 2 min fix)
```
? Uncheck/recheck Discord checkbox
? Switch mode and back
? Clear message box and retype
? Click DISCONNECT ? wait 5 sec ? CONNECT
```

### LEVEL 2: MODERATE ISSUE (2-5 min fix)
```
? Close and reopen app
? Check Discord is running
? Verify internet connection
? Test Discord website in browser
? Check bot still in server
```

### LEVEL 3: SERIOUS ISSUE (5-15 min fix)
```
? Go to Discord Developer Portal
? Check all 3 intents enabled
? Reset bot token
? Copy NEW token
? Update token in app
? Reconnect
```

### LEVEL 4: CRITICAL ISSUE (15+ min fix)
```
? Create entirely new bot application
? Enable all 3 intents FIRST
? Copy token immediately
? Invite to server
? Configure in app
? Test thoroughly
```

## ?? PRE-FLIGHT CHECKLIST

### Before Connecting First Time
```
? Bot created in Developer Portal
? All 3 privileged intents enabled
? Bot token copied and saved
? Bot invited to Discord server
? Bot visible in member list
? Channel ID copied from Discord
? Developer Mode enabled in Discord
```

### Before Each Connection Attempt
```
? Discord is running
? Internet is working
? Bot still in server
? Token is current (not expired)
? Channel ID is correct
? App is up to date
```

### After Successful Connection
```
? Status shows "Connected" (green)
? Send test message works
? Test message appears in Discord
? Discord messages appear in app
? User list loads (if testing DMs)
```

## ?? COMMON ERROR CODES

| Error Message | What It Means | Fix |
|--------------|---------------|-----|
| "Connection error" | Can't reach Discord | Check internet |
| "Invalid token" | Bot token wrong/expired | Reset token |
| "Channel not found" | Wrong channel ID | Copy ID again |
| "Missing intent: GuildMembers" | Intent not enabled | Enable + reset token |
| "Missing intent: MessageContent" | Intent not enabled | Enable + reset token |
| "Forbidden" | Bot lacks permissions | Check bot role |
| "Rate limited" | Too many requests | Wait 1 minute |
| "Gateway timeout" | Discord connectivity | Retry in 30 seconds |
| "Not connected" | Bot offline | Click CONNECT |
| "User not found" | Invalid user ID | Reselect user |

## ?? PRO TIPS

### Prevent Issues Before They Happen
```
? Enable all 3 intents BEFORE copying token
? Save token securely immediately
? Test connection before using
? Document your channel IDs
? Keep app updated
? Monitor Discord status page
? Restart app daily
? Check permissions monthly
```

### Speed Up Troubleshooting
```
? Always try simple fixes first
? Document what you tried
? Take screenshots of errors
? Note exact time of issues
? Check one thing at a time
? Don't change multiple things
? Test after each fix
? Keep training guide handy
```

### When to Reset Token
```
?? Connection fails repeatedly
?? After enabling new intents
?? Token possibly exposed
?? "Invalid token" errors
?? Bot shows as offline
?? Been > 90 days
?? Security best practice
```
