## ?? Common Mistakes Poster

```
?????????????????????????????????????????????????
?          ?? TOP 5 MISTAKES TO AVOID ??         ?
?????????????????????????????????????????????????

? #1: Forgetting Discord Checkbox
   ????????????????????????????????????????
   ? [Message typed]  ? Discord  [SEND]  ? ? WRONG!
   ????????????????????????????????????????
   ????????????????????????????????????????
   ? [Message typed]  ? Discord  [SEND]  ? ? RIGHT!
   ????????????????????????????????????????

? #2: Wrong Mode Selected
   Want channel ? But DM mode selected
   Want DM ? But channel mode selected
   ? CHECK BEFORE TYPING!

? #3: No User Selected in DM
   DM mode ? Type ? Send ? ERROR
   ? MUST SELECT USER FIRST!

? #4: Using Old Token
   Intents enabled ? Still failing
   ? RESET TOKEN AFTER ENABLING INTENTS!

? #5: Bot Not In Server
   Can't see users ? Bot not invited
   ? RE-INVITE WITH NEW PERMISSIONS!
```
