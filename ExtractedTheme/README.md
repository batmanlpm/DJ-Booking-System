## üöÄ Installation & Usage

### Step 1: Add Theme to Your Project

1. Copy `DJBookingTheme.xaml` to your WPF project
2. Place it in a `Themes` or `Resources` folder (recommended)

### Step 2: Merge Theme in App.xaml

```xml
<Application x:Class="YourApp.App"
             xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
             xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Application.Resources>
        <ResourceDictionary>
            <ResourceDictionary.MergedDictionaries>
                <ResourceDictionary Source="Themes/DJBookingTheme.xaml"/>
            </ResourceDictionary.MergedDictionaries>
        </ResourceDictionary>
    </Application.Resources>
</Application>
```

### Step 3: Apply Styles to Your Controls

```xml
<Window x:Class="YourApp.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Style="{StaticResource PrimaryWindowStyle}">
    
    <Grid>
        <Grid.Background>
            <StaticResource ResourceKey="SpaceBackgroundBrush"/>
        </Grid.Background>
        
        <Border Style="{StaticResource PanelBorderStyle}">
            <StackPanel>
                <TextBlock Text="Welcome to DJ Booking System"
                           Style="{StaticResource TitleTextBlockStyle}"/>
                
                <Button Content="Book Now"
                        Style="{StaticResource PrimaryButtonStyle}"/>
            </StackPanel>
        </Border>
    </Grid>
</Window>
```

## üé® Customization Guide

### Changing Primary Color

To change from neon green to another color:

1. Update the primary color definitions:
```xml
<Color x:Key="NeonGreenColor">#YOUR_COLOR</Color>
<Color x:Key="NeonGreenDarkColor">#YOUR_DARK_COLOR</Color>
```

2. Update glow effects:
```xml
<DropShadowEffect x:Key="NeonGreenGlowEffect" 
                  Color="{StaticResource NeonGreenColor}"/>
```

### Adjusting Animation Speed

Modify animation durations in storyboards:
```xml
<Storyboard x:Key="GentleFloatAnimation" RepeatBehavior="Forever">
    <DoubleAnimation Duration="0:0:3"/>  <!-- Change this value -->
</Storyboard>
```

### Customizing Spacing

Adjust padding and margin resources:
```xml
<Thickness x:Key="MediumPadding">15</Thickness>  <!-- Change from 10 to 15 -->
```

### Modifying Font Sizes

Update font size resources:
```xml
<System:Double x:Key="FontSizeNormal">16</System:Double>  <!-- Change from 14 -->
```

## üìã Best Practices

### 1. Consistent Styling
Always use the provided styles for consistency:
```xml
<!-- Good -->
<Button Style="{StaticResource PrimaryButtonStyle}"/>

<!-- Avoid -->
<Button Background="Green" Foreground="Black"/>
```

### 2. Proper Hierarchy
Use appropriate text styles for hierarchy:
```xml
<TextBlock Style="{StaticResource TitleTextBlockStyle}"/>      <!-- Main title -->
<TextBlock Style="{StaticResource SubtitleTextBlockStyle}"/>   <!-- Subtitle -->
<TextBlock Style="{StaticResource PrimaryTextBlockStyle}"/>    <!-- Body text -->
```

### 3. Spacing
Use consistent spacing resources:
```xml
<StackPanel>
    <Border Margin="{StaticResource MediumMargin}"/>
    <Border Padding="{StaticResource LargePadding}"/>
</StackPanel>
```

### 4. Color Usage
Use semantic color names:
```xml
<TextBlock Foreground="{StaticResource SuccessGreenBrush}"/>  <!-- For success -->
<TextBlock Foreground="{StaticResource ErrorRedBrush}"/>      <!-- For errors -->
<TextBlock Foreground="{StaticResource InfoCyanBrush}"/>      <!-- For info -->
```

### 5. Effects
Apply effects appropriately:
```xml
<!-- For important panels -->
<Border Effect="{StaticResource PanelShadowEffect}"/>

<!-- For cards -->
<Border Effect="{StaticResource CardShadowEffect}"/>

<!-- For focused elements -->
<Border Effect="{StaticResource NeonGreenGlowEffect}"/>
```

## üêõ Troubleshooting

### Theme Not Loading
**Problem**: Styles not applying to controls
**Solution**: Ensure theme is properly merged in App.xaml or Window resources

### Colors Not Showing
**Problem**: Colors appear as default system colors
**Solution**: Check that StaticResource references are correct and theme is loaded

### Animations Not Working
**Problem**: Animations don't play
**Solution**: Ensure RenderTransform is set on animated elements

### Performance Issues
**Problem**: Application runs slowly with theme
**Solution**: Reduce number of animated elements or simplify effects

## üìä Performance Considerations

### Optimization Tips

1. **Limit Animated Elements**: Don't animate too many elements simultaneously
2. **Use Hardware Acceleration**: Enable GPU rendering for better performance
3. **Simplify Effects**: Use lighter blur effects when possible
4. **Lazy Loading**: Load heavy visual elements only when needed

### Recommended Settings

```xml
<Window RenderOptions.BitmapScalingMode="HighQuality"
        RenderOptions.EdgeMode="Aliased"
        UseLayoutRounding="True"
        SnapsToDevicePixels="True">
```
