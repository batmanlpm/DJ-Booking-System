## ğŸ› Troubleshooting

### "No files found"
**Solutions:**
- Check drive letters are correct
- Verify file extensions are typed correctly
- Try searching without the dot (e.g., `mp3` instead of `.mp3`)
- Check if files actually exist

### "Access denied" errors
**Solutions:**
- Run program as Administrator
- Some system folders are protected
- Check file permissions

### "Search is slow"
**Solutions:**
- Select fewer drives
- Limit file types
- Exclude system folders (already automatic)

### Duplicate files when copying
**Solutions:**
- This is normal! Files auto-rename
- Check the folder for `filename_1.ext`, `filename_2.ext`, etc.
