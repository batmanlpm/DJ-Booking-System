## ?? How It Works

The tab styling uses **WPF Triggers** to dynamically change appearance:

1. **`IsSelected = True`** ? Red border, bigger size
2. **`IsSelected = False`** ? Blue border, smaller size
3. **`IsMouseOver = True`** (when not selected) ? Cyan highlight

### Size Difference:
- **Selected**: 18x10 padding, 14px font, 3px border
- **Unselected**: 14x7 padding, 12px font, 2px border

This creates a **visual hierarchy** where the active tab "pops out" with its red glow!

## ?? Customization Options

Want to adjust the design? Edit `Themes/CustomTabStyles.xaml`:

### Change Border Colors:
```xaml
<!-- Red border for selected -->
<Setter TargetName="Bd" Property="BorderBrush" Value="#FF0000"/>

<!-- Blue border for unselected -->
<Setter TargetName="Bd" Property="BorderBrush" Value="#0000FF"/>

<!-- Cyan border for hover -->
<Setter TargetName="Bd" Property="BorderBrush" Value="#00FFFF"/>
```

### Change Size Difference:
```xaml
<!-- Make selected tabs even bigger -->
<Setter Property="Padding" Value="20,12"/>  <!-- was 18,10 -->
<Setter Property="FontSize" Value="16"/>     <!-- was 14 -->

<!-- Make unselected tabs smaller -->
<Setter Property="Padding" Value="12,6"/>   <!-- was 14,7 -->
<Setter Property="FontSize" Value="11"/>    <!-- was 12 -->
```

### Change Glow Intensity:
```xaml
<!-- Stronger red glow -->
<DropShadowEffect Color="#FF0000" BlurRadius="30" ShadowDepth="0" Opacity="1.0"/>

<!-- Subtler blue glow -->
<DropShadowEffect Color="#0000FF" BlurRadius="5" ShadowDepth="0" Opacity="0.3"/>
```
