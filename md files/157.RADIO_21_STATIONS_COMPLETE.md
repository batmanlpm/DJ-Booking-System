## ?? **Adding More Stations**

### **For Users (In App)**
1. Click "Add Station" in Radio Player
2. Enter station name
3. Enter stream URL
4. Select genre
5. Click Save

### **For Developers (Code)**
Add to `InitializeDefaultStations()`:
```csharp
_stations.Add(new RadioStation
{
    Name = "Station Name",
    StreamUrl = "http://stream.url.here",
    Genre = "Genre Name",
    AddedBy = "System",
    AddedDate = DateTime.Now,
    IsFavorite = false // or true
});
```
