## ?? CODE EXAMPLE

### Using Shared Settings:

```csharp
// Get the singleton instance
var settings = CandyBotSharedSettings.Instance;

// Read settings
string personality = settings.CurrentPersonality;
bool voiceOn = settings.VoiceEnabled;
int speed = settings.SpeechRate;

// Change settings (automatically syncs & saves)
settings.CurrentPersonality = "Raunchy";
settings.VoiceEnabled = false;
settings.SpeechRate = 2;

// Subscribe to changes from other mode
settings.PropertyChanged += (s, e) => {
    if (e.PropertyName == nameof(CandyBotSharedSettings.CurrentPersonality))
    {
        UpdatePersonalityUI();
    }
};
```
