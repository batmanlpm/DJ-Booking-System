# ?? Quick Integration Script - RadioBoss & Features

## ? **5-Minute Integration**

Follow these steps to quickly integrate RadioBoss views:

---

## **Step 1: Add to MainWindow.xaml**

### **1A: Add namespace at top of MainWindow.xaml**
```xaml
xmlns:radio="clr-namespace:DJBookingSystem.Views.Radio"
```

### **1B: Add menu items after existing Radio menu**
Find the menu bar and add:
```xaml
<!-- After existing Radio menu -->
<MenuItem Header="RadioBoss Control" Style="{StaticResource HorizontalMenuItemStyle}">
    <MenuItem Header="?? Cloud Control" Click="Menu_RadioBossCloud_Click"/>
    <MenuItem Header="?? Stream Control" Click="Menu_RadioBossStream_Click"/>
</MenuItem>
```

### **1C: Add panels in content area**
Find where other panels are (BookingsPanel, VenuesPanel, etc.) and add:
```xaml
<!-- RadioBoss Cloud Panel -->
<Grid x:Name="RadioBossCloudPanel" Visibility="Collapsed">
    <radio:RadioBossCloudView/>
</Grid>

<!-- RadioBoss Stream Panel -->
<Grid x:Name="RadioBossStreamPanel" Visibility="Collapsed">
    <radio:RadioBossStreamView/>
</Grid>
```

---

## **Step 2: Add to MainWindow.MenuHandlers.cs**

### **2A: Update HideAllPanels method**
Add these two lines to `HideAllPanels()`:
```csharp
if (RadioBossCloudPanel != null) RadioBossCloudPanel.Visibility = Visibility.Collapsed;
if (RadioBossStreamPanel != null) RadioBossStreamPanel.Visibility = Visibility.Collapsed;
```

### **2B: Add menu click handlers**
Add anywhere in the `#region Menu Click Handlers` section:
```csharp
private void Menu_RadioBossCloud_Click(object sender, RoutedEventArgs e)
{
    ShowPanel(RadioBossCloudPanel);
}

private void Menu_RadioBossStream_Click(object sender, RoutedEventArgs e)
{
    ShowPanel(RadioBossStreamPanel);
}
```

## **Troubleshooting**

### **Problem: Can't see RadioBoss menu**
- Check namespace is added: `xmlns:radio="..."`
- Check files exist in Views/Radio/ folder
- Rebuild solution

### **Problem: Auto-login not working**
- WebBrowser control has limited JavaScript support
- Consider upgrading to WebView2 for better auto-login
- For now, manually login when page loads

### **Problem: Full screen not working**
- Check parent window is found in Loaded event
- Verify button click handler is wired up
- Check window style changes

### **Problem: Page won't load**
- Check internet connection
- Verify URL is correct
- Check firewall isn't blocking WebBrowser

## **Optional: Add Button Style**

If ModernButtonStyle doesn't exist, add to your styles:

```xaml
<Style x:Key="ModernButtonStyle" TargetType="Button">
    <Setter Property="Background" Value="Transparent"/>
    <Setter Property="Foreground" Value="#00FF00"/>
    <Setter Property="BorderBrush" Value="#00FF00"/>
    <Setter Property="BorderThickness" Value="2"/>
    <Setter Property="FontFamily" Value="Consolas"/>
    <Setter Property="FontWeight" Value="Bold"/>
    <Setter Property="Cursor" Value="Hand"/>
    <Setter Property="Template">
        <Setter.Value>
            <ControlTemplate TargetType="Button">
                <Border Background="{TemplateBinding Background}"
                       BorderBrush="{TemplateBinding BorderBrush}"
                       BorderThickness="{TemplateBinding BorderThickness}"
                       CornerRadius="5"
                       Padding="10,5">
                    <ContentPresenter HorizontalAlignment="Center" 
                                    VerticalAlignment="Center"/>
                </Border>
            </ControlTemplate>
        </Setter.Value>
    </Setter>
    <Style.Triggers>
        <Trigger Property="IsMouseOver" Value="True">
            <Setter Property="Background" Value="#00FF00"/>
            <Setter Property="Foreground" Value="#000000"/>
        </Trigger>
    </Style.Triggers>
</Style>
```
