## ?? USER FLOW

### Opening Settings:

```
1. User Right-Clicks Widget
        ?
2. Context Menu Appears
        ?
3. User Clicks "Settings"
        ?
4. Settings Window Opens
        ?
5. Personality Tab Shows (Default)
        ?
6. User Can Switch Tabs
        ?
7. Changes Apply Immediately
        ?
8. User Closes Window
```

### Changing Personality:

```
1. Open Settings
        ?
2. Already on Personality Tab
        ?
3. Click Different Personality
        ?
4. Voice Says "Personality changed!"
        ?
5. New Personality Active
```

### Adjusting Voice:

```
1. Open Settings
        ?
2. Click "Audio" Tab
        ?
3. Toggle Voice On/Off
        ?
4. Adjust Speech Rate Slider
        ?
5. Click "Test Voice"
        ?
6. Hear Voice Sample
```
