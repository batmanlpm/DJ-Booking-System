# ?? Candy-Bot Voice Recording Guide ??

## ?? **Audio Format Requirements**

### **Recommended Format**
- **File Type**: `.wav` or `.mp3`
- **Sample Rate**: 44.1 kHz or 48 kHz (CD quality)
- **Bit Depth**: 16-bit or 24-bit
- **Channels**: Mono (single voice) preferred
- **Bitrate**: 128 kbps minimum (for MP3)

### **Technical Specifications**
```
Format: WAV (uncompressed) or MP3 (compressed)
- WAV: Best quality, larger files (~10MB per minute)
- MP3: Good quality, smaller files (~1MB per minute)

Sampling: 44100 Hz (44.1 kHz) - Standard CD quality
Bit Depth: 16-bit for delivery, 24-bit for recording
Mono: Single channel (one voice)
```

### **File Naming Convention**
```
candybot_[category]_[number]_[description].wav

Examples:
- candybot_greeting_01_hello.wav
- candybot_help_05_booking_question.wav
- candybot_flirt_03_wink_response.wav
- candybot_thanks_02_appreciation.wav
```

## ??? **Recording Session Structure**

### **Session Setup**
- **Quiet room** with minimal echo
- **Pop filter** to reduce plosives (P, B sounds)
- **Good microphone** (USB condenser mic works great)
- **Recording software**: Audacity (free) or Adobe Audition

### **Recording Tips**
1. **Warm up** your voice (hum, scales, lip trills)
2. **Smile while recording** - it changes the tone!
3. **Stay hydrated** - keep water nearby
4. **Take breaks** every 15-20 phrases
5. **Record multiple takes** of each phrase
6. **Stay consistent** in distance from mic
7. **Avoid mouth clicks** - drink water, avoid dairy before recording

### **Post-Processing**
```
1. Normalize audio to -3dB
2. Remove noise (if any)
3. Trim silence at start/end
4. Add 0.1s fade in/out
5. Compress dynamics slightly
6. Equalize for clarity (boost 2-5kHz range)
7. Export as WAV (44.1kHz, 16-bit, Mono)
```

## ?? **Implementation in C#**

### **Updated CandyBotSoundManager.cs**

```csharp
using System;
using System.IO;
using System.Media;
using System.Windows.Media;

namespace DJBookingSystem.Services
{
    public class CandyBotSoundManager : IDisposable
    {
        private MediaPlayer _mediaPlayer;
        private bool _voiceEnabled = false;
        private bool _soundsEnabled = true;
        private string _voiceDirectory = "Voices/CandyBot/";

        public CandyBotSoundManager()
        {
            _mediaPlayer = new MediaPlayer();
        }

        /// <summary>
        /// Enable or disable voice mode
        /// </summary>
        public void SetVoiceMode(bool enabled)
        {
            _voiceEnabled = enabled;
        }

        /// <summary>
        /// Enable or disable sound effects
        /// </summary>
        public void SetSoundsEnabled(bool enabled)
        {
            _soundsEnabled = enabled;
        }

        /// <summary>
        /// Play a voice line from Candy-Bot
        /// </summary>
        public void PlayVoiceLine(string category, string filename)
        {
            if (!_voiceEnabled) return;

            try
            {
                string fullPath = Path.Combine(_voiceDirectory, category, filename);
                
                if (File.Exists(fullPath))
                {
                    _mediaPlayer.Open(new Uri(fullPath, UriKind.Relative));
                    _mediaPlayer.Play();
                }
                else
                {
                    System.Diagnostics.Debug.WriteLine($"Voice file not found: {fullPath}");
                }
            }
            catch (Exception ex)
            {
                System.Diagnostics.Debug.WriteLine($"Error playing voice: {ex.Message}");
            }
        }

        /// <summary>
        /// Play a greeting voice line
        /// </summary>
        public void PlayGreeting()
        {
            string[] greetings = {
                "candybot_greeting_01_hello.wav",
                "candybot_greeting_02_welcome.wav",
                "candybot_greeting_03_hi_there.wav"
            };
            
            var random = new Random();
            string greeting = greetings[random.Next(greetings.Length)];
            PlayVoiceLine("Greetings", greeting);
        }

        /// <summary>
        /// Play a help voice line
        /// </summary>
        public void PlayHelpResponse()
        {
            PlayVoiceLine("Help", "candybot_help_01_assistance.wav");
        }

        /// <summary>
        /// Play a positive feedback voice line
        /// </summary>
        public void PlayPositiveFeedback()
        {
            string[] responses = {
                "candybot_positive_01_awesome.wav",
                "candybot_positive_02_perfect.wav",
                "candybot_positive_03_great.wav"
            };
            
            var random = new Random();
            string response = responses[random.Next(responses.Length)];
            PlayVoiceLine("Positive", response);
        }

        /// <summary>
        /// Play an error/apology voice line
        /// </summary>
        public void PlayErrorResponse()
        {
            PlayVoiceLine("Errors", "candybot_error_01_oops.wav");
        }

        /// <summary>
        /// Play sound effect
        /// </summary>
        public void PlaySoundEffect(string effectName)
        {
            if (!_soundsEnabled) return;

            try
            {
                // Play system sounds or custom sound effects
                switch (effectName.ToLower())
                {
                    case "notification":
                        SystemSounds.Beep.Play();
                        break;
                    case "error":
                        SystemSounds.Hand.Play();
                        break;
                    case "success":
                        SystemSounds.Asterisk.Play();
                        break;
                }
            }
            catch (Exception ex)
            {
                System.Diagnostics.Debug.WriteLine($"Error playing sound: {ex.Message}");
            }
        }

        public void Dispose()
        {
            _mediaPlayer?.Close();
        }
    }
}
```

## ?? **Pro Tips for Voice Acting**

1. **Smile for sweetness** - Literally smile while recording happy phrases
2. **Lower pitch slightly** for "serious" or "sultry" phrases
3. **Speak slightly slower** than normal conversation
4. **Emphasize key words** - "I'm SOO excited to help!"
5. **Add natural pauses** - "Hi there! *pause* How can I help?"
6. **Vary your delivery** across takes
7. **Stay in character** for each personality mode
8. **Have fun!** Enthusiasm shows in the voice
