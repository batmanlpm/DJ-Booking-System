## Quick Fix Command

To replace emojis programmatically:

```powershell
# Find all files with emojis
Get-ChildItem -Recurse -Include *.xaml,*.cs | 
  ForEach-Object {
    $content = Get-Content $_.FullName -Raw -Encoding UTF8
    if ($content -match '[^\x00-\x7F]') {
      Write-Host $_.FullName
    }
  }
```
