    }

## ğŸ› Troubleshooting

### Image Generation Issues

**Problem:** "Invalid API key"
**Solution:** Get API key from https://platform.openai.com/api-keys

**Problem:** "Rate limit exceeded"
**Solution:** Wait a minute or upgrade OpenAI plan

**Problem:** "Content policy violation"
**Solution:** Revise prompt to be more appropriate

### Document Generation Issues

**Problem:** "Text file created instead of Word doc"
**Solution:** Install DocumentFormat.OpenXml package

**Problem:** "Can't open generated file"
**Solution:** Install Microsoft Office or compatible viewer

**Problem:** "File already exists"
**Solution:** Files are timestamped, check output folder
