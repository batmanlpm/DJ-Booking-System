### **Step 2: Register the Controls Namespace**

At the top of `MainWindow.xaml`, add:

```xml
xmlns:controls="clr-namespace:DJBookingSystem.Controls"
```

### **Step 3: Update Connection Status Programmatically**

In your `MainWindow.xaml.cs`, add methods to update indicator status:

```csharp
private void UpdateClaudeIndicators(string status)
{
    // Find all ClaudeIndicator controls in the window
    var indicators = FindVisualChildren<ClaudeIndicator>(this);
    foreach (var indicator in indicators)
    {
        indicator.ConnectionStatus = status;
    }
}

// Helper method to find child controls
private static IEnumerable<T> FindVisualChildren<T>(DependencyObject depObj) where T : DependencyObject
{
    if (depObj != null)
    {
        for (int i = 0; i < VisualTreeHelper.GetChildrenCount(depObj); i++)
        {
            DependencyObject child = VisualTreeHelper.GetChild(depObj, i);
            if (child != null && child is T)
            {
                yield return (T)child;
            }

            foreach (T childOfChild in FindVisualChildren<T>(child))
            {
                yield return childOfChild;
            }
        }
    }
}
```

## 游꿛 CUSTOMIZATION OPTIONS

### **Indicator Colors**
You can customize the indicator color for different purposes:

```xml
<!-- Database Connection Indicator -->
<controls:ClaudeIndicator Stroke="#0088FF"/> <!-- Blue for database -->

<!-- API Connection Indicator -->
<controls:ClaudeIndicator Stroke="#FF8800"/> <!-- Orange for API -->

<!-- Discord Connection Indicator -->
<controls:ClaudeIndicator Stroke="#5865F2"/> <!-- Discord purple -->
```

### **Animation Speed**
Adjust the glow animation speed in `ClaudeIndicator.xaml`:

```xml
<DoubleAnimation From="0.4" To="1.0" Duration="0:0:1.5"/> <!-- Slower -->
<DoubleAnimation From="0.4" To="1.0" Duration="0:0:0.5"/> <!-- Faster -->
```

### **Connection Status Colors**
The indicator automatically adjusts based on `ConnectionStatus` property:
- **Connected**: Normal green glow (speed 1.0)
- **Connecting**: Faster green pulse (speed 2.0)
- **Disconnected**: Slower dim pulse (speed 0.5)
