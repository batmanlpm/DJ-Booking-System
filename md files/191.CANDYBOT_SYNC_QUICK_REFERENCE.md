## ?? FOR DEVELOPERS

### Get Settings Instance:
```csharp
var settings = CandyBotSharedSettings.Instance;
```

### Read a Setting:
```csharp
string personality = settings.CurrentPersonality;
bool voiceOn = settings.VoiceEnabled;
```

### Change a Setting:
```csharp
settings.CurrentPersonality = "Funny";
settings.VoiceEnabled = false;
// Automatically syncs & saves!
```

### Subscribe to Changes:
```csharp
settings.PropertyChanged += (s, e) => {
    // Update UI when other mode changes
    UpdateMyUI();
};
```
