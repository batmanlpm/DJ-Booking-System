## ?? **HOW TO USE**

### **Access**
```
Main Menu ? Users ? Create User
```

### **Steps**
```
1. Enter Username
2. Enter Password
3. Confirm Password
4. Select Role:
   - User
   - DJ
   - VenueOwner
   - Manager (SysAdmin only)
   - SysAdmin (SysAdmin only)
5. Click "CREATE USER"
```

### **Auto-Generated**
The following are created automatically:
- **Full Name**: Same as username
- **Email**: username@djbooking.local
- **Permissions**: Based on role
- **Default Theme**: Green
- **Account Status**: Active
