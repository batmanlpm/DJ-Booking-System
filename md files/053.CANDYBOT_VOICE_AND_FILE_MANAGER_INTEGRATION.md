## üìÅ File Management Features

### Security First Design

**ALL file operations require user approval via UI**

```csharp
var fileManager = new CandyBotFileManager();

// User must approve folders first
fileManager.AddApprovedFolder(@"C:\Users\GOD\Music");
fileManager.AddApprovedFolder(@"K:\Customer Data\LPM");

// Unapproved access is automatically denied
```

### File Search

```csharp
// Search for all files
var allFiles = await fileManager.SearchFilesAsync();

// Search specific extension
var mp3Files = await fileManager.SearchFilesAsync(
    fileExtension: "mp3",
    includeSubfolders: true
);

// Search in specific folder
var docs = await fileManager.SearchFilesAsync(
    folderPath: @"C:\Users\GOD\Documents"
);

// Search file content
var matches = await fileManager.SearchFileContentAsync(
    searchText: "DJ Booking",
    extensions: new[] { ".txt", ".md", ".cs" }
);
```

### File Operations

```csharp
// Open file with default app
bool opened = fileManager.OpenFile(@"C:\Users\GOD\setlist.txt");

// Copy file
bool copied = fileManager.CopyFile(
    sourcePath: @"C:\Music\track.mp3",
    destinationPath: @"C:\DJ Sets\track.mp3",
    overwrite: false
);

// Move file
bool moved = fileManager.MoveFile(
    sourcePath: @"C:\Downloads\mix.mp3",
    destinationPath: @"C:\Music\mix.mp3"
);
```

### DJ & Media Management

```csharp
// Find all music files
var musicFiles = await fileManager.FindMusicFilesAsync();
// Returns: .mp3, .wav, .flac, .m4a, .aac, .ogg

// Find DJ setlists
var setlists = await fileManager.FindDJSetlistsAsync();
// Returns: Files with "setlist" or "playlist" in name

// Auto-organize files by type
int organized = await fileManager.OrganizeFilesByTypeAsync(
    @"C:\Downloads"
);
// Creates: /Music, /Documents, /Images, /Videos folders
```

### Document Analysis

```csharp
// Read text file
string content = await fileManager.ReadTextFileAsync(@"C:\notes.txt");

{
    Console.WriteLine($"{type.Key}: {type.Value} files");
}
```

### Audit Trail

### Example 2: DJ File Assistant

```csharp
public class DJFileAssistant
{
    private CandyBotFileManager _files;
    private CandyBotSoundManager _sound;
    
    public async Task OrganizeMusicLibrary(string musicFolder)
    {
        // Initialize
        _files = new CandyBotFileManager();
        _sound = new CandyBotSoundManager();
        _sound.SetVoiceMode(true);
        
        // Request permission
        _files.AddApprovedFolder(musicFolder);
        
        // Find all music
        await _sound.PlayVoiceLine("033"); // "Looking for available venues?"
        var musicFiles = await _files.FindMusicFilesAsync(musicFolder);
        
        // Organize
        await _sound.PlayVoiceLine("032"); // "Let's get that DJ booked!"
        int organized = await _files.OrganizeFilesByTypeAsync(musicFolder);
        
        // Success!
        await _sound.PlayVoiceLine("048"); // "Boom! Task completed!"
        MessageBox.Show($"Organized {organized} music files!");
    }
}
```

### Example 3: Setlist Finder with Context

```csharp
public class SetlistFinder
{
    public async Task FindAndOpenSetlist(string searchTerm)
    {
        var files = new CandyBotFileManager();
        var sound = new CandyBotSoundManager();
        sound.SetVoiceMode(true);
        
        // Search
        await sound.PlayVoiceLine("018"); // "Looking for something specific?"
        
        var setlists = await files.FindDJSetlistsAsync();
        var matches = setlists.Where(s => 
            s.FileName.Contains(searchTerm, StringComparison.OrdinalIgnoreCase));
        
        if (matches.Any())
        {
            await sound.PlayVoiceLine("042"); // "Perfect! You're all set!"
            files.OpenFile(matches.First().FilePath);
        }
        else
        {
            await sound.PlayVoiceLine("058"); // "Hmm, I couldn't find that"
        }
    }
}
```

## üîß Troubleshooting

### Voice Not Playing

1. **Check voice mode enabled**
   ```csharp
   soundManager.SetVoiceMode(true);
   soundManager.SetSoundsEnabled(true);
   ```

2. **Verify files exist**
   ```csharp
   List<string> missing;
   CandyBotVoiceMapper.VerifyVoiceFiles(out missing);
   ```

3. **Check file paths**
   - Voice files must be in `/Voices/` folder
   - Files named 001.mp3 - 100.mp3

### File Operations Fail

1. **Folder not approved**
   ```csharp
   fileManager.AddApprovedFolder(folderPath);
   ```

2. **Check operation log**
   ```csharp
   var log = fileManager.GetOperationLog();
   // Look for "Access denied" messages
   ```

3. **Verify permissions**
   - Windows file permissions
   - Folder exists and accessible
