## ?? Design Consistency Guidelines

All windows should follow this pattern:

### Colors
- **Background**: `#000000` (Pure Black)
- **Secondary Background**: `#0A0A0A` (Very Dark Gray)
- **Primary Accent**: `#00FF00` (Neon Green)
- **Borders**: `#00FF00` (Neon Green, 2px)
- **Text**: `#00FF00` (Neon Green)
- **Secondary Text**: `#888888` or `#666666` (Gray)
- **Success**: `#27AE60` (Green)
- **Error**: `#E74C3C` (Red)

### Fonts
- **Primary**: `Consolas` (monospace, futuristic)
- **Titles**: Size 20-28, Bold
- **Body**: Size 13-14, Normal
- **Small**: Size 10-12, Italic for hints

### Effects
- **Glow**: `DropShadowEffect` with `#00FF00`, BlurRadius 15-25
- **Borders**: CornerRadius 4-8 for modern rounded corners
- **Buttons**: Bold, uppercase text with glow effects

### Window Structure
```xaml
<Window WindowStyle="None" AllowsTransparency="True" Background="Black">
    <Border BorderBrush="#00FF00" BorderThickness="2">
        <Border.Effect>
            <DropShadowEffect Color="#00FF00" BlurRadius="20"/>
        </Border.Effect>
        <Grid>
            <!-- Title Bar (Grid.Row="0") -->
            <!-- Content (Grid.Row="1") -->
            <!-- Buttons (Grid.Row="2") -->
        </Grid>
    </Border>
</Window>
```
