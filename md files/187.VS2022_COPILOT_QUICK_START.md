## Copilot-Ready Code Templates

### Template 1: Voice-Enabled Button

**Ask Copilot:** "Create a button that plays a random greeting"

Copilot will suggest:
```csharp
private CandyBotSoundManager _soundManager = new CandyBotSoundManager();

private async void GreetButton_Click(object sender, RoutedEventArgs e)
{
    _soundManager.SetVoiceMode(true);
    await _soundManager.PlayRandomVoiceFromCategory("Greeting");
}
```

### Template 2: File Search with Voice Feedback

**Ask Copilot:** "Search for music files with voice feedback"

Copilot will suggest:
```csharp
private async void SearchMusic()
{
    var sound = new CandyBotSoundManager();
    var files = new CandyBotFileManager();
    
    sound.SetVoiceMode(true);
    
    // Tell user we're searching
    await sound.PlayVoiceLine("018"); // "Looking for something specific?"
    
    // Approve folder
    files.AddApprovedFolder(@"C:\Users\Music");
    
    // Search
    var musicFiles = await files.FindMusicFilesAsync();
    
    if (musicFiles.Count > 0)
    {
        await sound.PlayVoiceLine("042"); // "Perfect!"
        ResultsList.ItemsSource = musicFiles;
    }
    else
    {
        await sound.PlayVoiceLine("058"); // "Couldn't find that"
    }
}
```

### Template 3: Organize Files with Progress

**Ask Copilot:** "Organize files with voice progress updates"

Copilot will suggest:
```csharp
private async void OrganizeFiles_Click(object sender, RoutedEventArgs e)
{
    var sound = new CandyBotSoundManager();
    var files = new CandyBotFileManager();
    
    sound.SetVoiceMode(true);
    
    // Start
    await sound.PlayVoiceLine("032"); // "Let's get that DJ booked!"
    
    // Organize
    int organized = await files.OrganizeFilesByTypeAsync(@"C:\Downloads");
    
    // Success
    await sound.PlayVoiceLine("048"); // "Boom! Task completed!"
    MessageBox.Show($"Organized {organized} files!");
}
```
