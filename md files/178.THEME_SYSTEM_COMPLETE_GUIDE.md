## ?? How to Use

### In Code (C#)

```csharp
// Apply a theme
ThemeManager.ApplyColorTheme("Green");    // Matrix Green
ThemeManager.ApplyColorTheme("Blue");     // Cyber Blue
ThemeManager.ApplyColorTheme("Rainbow");  // Animated Rainbow

// Get theme display name
string name = ThemeManager.GetThemeDisplayName("Rainbow");
// Returns: "?? Rainbow (Animated)"

// Get preview color for UI
Color preview = ThemeManager.GetThemePreviewColor("Purple");
// Returns: RGB(180, 0, 255)

// Check current theme
string current = ThemeManager.CurrentTheme;

// Check if Rainbow is active
bool rainbowActive = ThemeManager.IsRainbowActive;
```

### In XAML

```xaml
<!-- Use dynamic resources that automatically update -->
<TextBlock Foreground="{DynamicResource AccentBrush}" Text="Hello"/>

<!-- Colors update when theme changes -->
<Border BorderBrush="{DynamicResource AccentBrush}" BorderThickness="2"/>

<!-- Glow effects automatically use current theme color -->
<TextBlock.Effect>
    <DropShadowEffect Color="{DynamicResource AccentColor}" 
                      BlurRadius="15" 
                      ShadowDepth="0"/>
</TextBlock.Effect>
```

## ?? Rainbow Theme Special Features

### Animation Characteristics
- **Update Rate**: 50ms (20 FPS)
- **Hue Increment**: 2° per frame
- **Full Cycle**: ~9 seconds (360° / 2° × 50ms)
- **Color Space**: HSV (Hue, Saturation, Value)
- **Saturation**: 100% (fully vibrant)
- **Brightness**: 100% (full intensity)

### Performance
- **CPU Usage**: < 1% on modern systems
- **Memory**: Negligible increase
- **GPU**: Hardware-accelerated WPF rendering
- **Animation Type**: Smooth color transitions with easing

### Control Methods
```csharp
// Start animation
RainbowThemeAnimator.StartAnimation();

// Stop animation
RainbowThemeAnimator.StopAnimation();

// Check status
bool isRunning = RainbowThemeAnimator.IsAnimating;

// Adjust speed (1.0 = normal, 2.0 = double speed)
RainbowThemeAnimator.SetAnimationSpeed(2.0);
```

## ?? Technical Architecture

### Resource Hierarchy

```
App.xaml
  ??? Application.Resources
       ??? MergedDictionaries
            ??? SpaceTheme.xaml           (Base styles)
            ??? [ColorTheme].xaml         (Active color)
            ??? CustomTabStyles.xaml      (Tab styling)
```

### Dynamic Resource Binding

All UI elements use `{DynamicResource}` binding:

```xaml
<!-- Automatically updates when theme changes -->
Foreground="{DynamicResource AccentBrush}"
BorderBrush="{DynamicResource AccentBrush}"
Color="{DynamicResource AccentColor}"
```

### Theme Switching Process

1. **Remove old theme** from MergedDictionaries
2. **Load new theme** XAML file
3. **Add to resources** at runtime
4. **WPF automatically re-evaluates** all DynamicResource bindings
5. **UI updates instantly** without reloading
