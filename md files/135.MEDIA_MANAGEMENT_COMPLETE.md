```csharp

    bucketName: "your-bucket",
    accessKey: "YOUR_ACCESS_KEY",
    secretKey: "YOUR_SECRET_KEY",
    region: "us-east-1"

```

## üöÄ Usage Examples

### Enhanced File Organizer:
```csharp
var organizer = new EnhancedFileOrganizerService(videoService, s3Service);

var results = await organizer.OrganizeWithMetadataAsync(
    sourcePath: @"C:\Downloads",
    sortByResolution: true,
    extractMetadata: true,
    uploadToS3: true
);
```

### Schedule Content:
```csharp
var scheduler = new ContentSchedulerService(s3Service);

await scheduler.SchedulePostWithUploadAsync(
    title: "New Video",
    description: "Latest content",
    scheduledTime: DateTime.Now.AddHours(2),
    filePath: @"C:\video.mp4",
    platform: "Website"
);
```

### Customer Service Bot:
```csharp
var bot = new CustomerServiceBot();
var response = await bot.GetResponseAsync("How do I cancel subscription?");
Console.WriteLine(response.Response);
```
