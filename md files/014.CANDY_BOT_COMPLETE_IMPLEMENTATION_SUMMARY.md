## üìÅ File Manager Features

### Security-First Design
- ‚úÖ **Folder approval required** - User controls what Candy-Bot can access
- ‚úÖ **Operation logging** - Full audit trail of all file operations
- ‚úÖ **100% local** - Files never leave user's PC
- ‚úÖ **No telemetry** - Zero data collection

### Capabilities

**File Search:**
```csharp
var fileManager = new CandyBotFileManager();
fileManager.AddApprovedFolder(@"C:\Music");

// Search files
var mp3Files = await fileManager.SearchFilesAsync(fileExtension: "mp3");

// Search content
var matches = await fileManager.SearchFileContentAsync("DJ Booking");

// Find music
var musicFiles = await fileManager.FindMusicFilesAsync();
```

**File Operations:**
```csharp
// Open file
fileManager.OpenFile(@"C:\setlist.txt");

// Copy file
fileManager.CopyFile(source, destination);

// Move file
fileManager.MoveFile(source, destination);
```

**DJ/Media Management:**
```csharp
// Find all music files
var musicFiles = await fileManager.FindMusicFilesAsync();
// Returns: .mp3, .wav, .flac, .m4a, .aac, .ogg

// Find DJ setlists
var setlists = await fileManager.FindDJSetlistsAsync();

// Auto-organize
int organized = await fileManager.OrganizeFilesByTypeAsync(@"C:\Downloads");
// Creates: /Music, /Documents, /Images, /Videos
```

## üîß Troubleshooting

### Voice Not Playing?

1. Check voice mode enabled:
   ```csharp
   soundManager.SetVoiceMode(true);
   soundManager.SetSoundsEnabled(true);
   ```

2. Verify files exist:
   ```csharp
   List<string> missing;
   CandyBotVoiceMapper.VerifyVoiceFiles(out missing);
   ```

3. Check file paths - Files should be in `/Voices/` folder

### File Operations Failing?

1. Folder not approved:
   ```csharp
   fileManager.AddApprovedFolder(folderPath);
   ```

2. Check operation log:
   ```csharp
   var log = fileManager.GetOperationLog();
   // Look for "Access denied" messages
   ```
