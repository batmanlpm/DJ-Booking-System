## ğŸ› Troubleshooting

### "Voice not working"
**Solution:**
- Check Windows Speech is installed
- Go to: Settings > Time & Language > Speech
- Install a voice if none available

### "Can't find MultiDriveSearchWindow"
**Solution:**
- Make sure all files are in project
- Right-click project > Add > Existing Item
- Add the .xaml and .xaml.cs files

### "Search is slow"
**Solution:**
- Don't search all drives at once (select specific ones)
- Use `includeSubfolders: false` for faster results
- Add more excluded folders
