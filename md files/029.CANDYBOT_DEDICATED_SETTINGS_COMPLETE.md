## ?? HOW TO USE

### As a User:
1. **Right-click** Candy-Bot widget
2. **Click "Settings"**
3. **Choose a tab** to configure that category
4. **Make changes** - they apply immediately
5. **Close window** when done

### As a Developer:
```csharp
// Open settings window
var settings = new CandyBotSettingsWindow();
settings.Show();

// Open settings to specific tab
var settings = new CandyBotSettingsWindow();
settings.PersonalityTab.IsChecked = true; // Opens to Personality tab
settings.Show();
```
