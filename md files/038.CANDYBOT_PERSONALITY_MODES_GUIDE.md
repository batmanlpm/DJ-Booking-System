## ?? **HOW TO CHANGE PERSONALITY MODE**

### In Settings (Future Implementation):

1. Click **File ? My Settings**
2. Navigate to **Candy-Bot Preferences**
3. Select desired personality from dropdown:
   - ?? Normal
   - ?? Shy
   - ?? Funny
   - ?? Shit-Stirring
   - ?? Raunchy (18+)
   - ?? Professional
4. Click **Save**
5. Candy-Bot instantly adopts new personality!

### Via Code (For Development):

```csharp
// Set personality mode
var candyBotService = new CandyBotService(apiKey);
candyBotService.SetPersonality(CandyBotPersonalityMode.Funny);

// Or from user preferences
_preferences.PersonalityMode = CandyBotPersonalityMode.ShitStirring;
candyBotService.SetPersonality(_preferences.PersonalityMode);
```
