# ?? Claude Indicator with Service Labels

## Overview

The ClaudeIndicator control now displays a **label text** beneath each Claude face to identify which service it's monitoring:

- **Cosmos** - Azure Cosmos DB connection
- **LPM.fm** - Live Party Music website
- **Booking** - RadioBOSS control panel
- **Live Status** - RadioBOSS stream status

## Properties

### LabelText
- **Type**: `string`
- **Purpose**: Short label displayed under the Claude face
- **Examples**: 
  - `"Cosmos"` for Azure Cosmos DB
  - `"LPM.fm"` for Live Party Music website
  - `"Booking"` for RadioBOSS control panel
  - `"Live Status"` for stream status

### ServiceName
- **Type**: `string`
- **Purpose**: Full service name shown in tooltip
- **Examples**: 
  - `"Azure Cosmos DB"`
  - `"Live Party Music Website"`
  - `"RadioBOSS Control Panel"`
  - `"RadioBOSS Stream Status"`

### IsConnected
- **Type**: `bool`
- **Purpose**: Controls the color (green = true, red = false)
- **Default**: `false`

---

## Color States

### ?? Green (Connected)
- **IsConnected**: `true`
- **Label Color**: Green (#00FF00)
- **Face**: Smiling ??
- **Glow**: Normal speed

### ?? Red (Disconnected)
- **IsConnected**: `false`
- **Label Color**: Red (#FF0000)
- **Face**: Frowning ??
- **Glow**: Slow speed

### ?? Yellow (Connecting)
- **Method**: Call `SetConnecting()`
- **Label Color**: Yellow (#FFFF00)
- **Face**: Neutral
- **Glow**: Fast speed

## Code Examples

### Setting Connection Status

```csharp
// Set to connected (green)
CosmosIndicator.IsConnected = true;

// Set to disconnected (red)
LPMIndicator.IsConnected = false;

// Set to connecting (yellow)
RadioBossIndicator.SetConnecting();
```

### Checking All Connections

```csharp
// Check all 4 services
var cosmosOk = await ConnectionService.CheckCosmosDbAsync(cosmosService);
var lpmOk = await ConnectionService.CheckLivePartyMusicAsync();
var radioBossOk = await ConnectionService.CheckRadioBossControlAsync();
var streamOk = await ConnectionService.CheckRadioBossStreamStatusAsync();

// Update indicators
CosmosIndicator.IsConnected = cosmosOk;
LPMIndicator.IsConnected = lpmOk;
RadioBossIndicator.IsConnected = radioBossOk;
StreamIndicator.IsConnected = streamOk;
```

## Styling

### Label Font

The label uses:
- **Font Family**: Consolas (monospace)
- **Font Size**: 10pt
- **Font Weight**: Bold
- **Color**: Matches Claude's status color
- **Glow Effect**: 5px blur, same color as Claude

### Customizing Label Style

```xaml
<!-- In ClaudeIndicator.xaml, the label is defined as: -->
<TextBlock Text="{Binding LabelText, ...}"
           Foreground="{Binding StatusColor, ...}"
           FontFamily="Consolas"
           FontSize="10"
           FontWeight="Bold"
           HorizontalAlignment="Center"
           Margin="0,3,0,0">
    <TextBlock.Effect>
        <DropShadowEffect Color="{Binding StatusColorValue, ...}" 
                        BlurRadius="5" 
                        ShadowDepth="0"
                        Opacity="0.6"/>
    </TextBlock.Effect>
</TextBlock>
```
