## ?? TROUBLESHOOTING

### "Context menu doesn't show new items"
**Solution:**
- Rebuild solution (Ctrl+Shift+B)
- Close and restart application
- Clear WPF cache

### "Multi-Drive Search window won't open"
**Solution:**
- Check MultiDriveSearchWindow.xaml exists
- Verify using statements in CandyBotAvatar.xaml.cs
- Check error output window

### "Quick search does nothing"
**Solution:**
- Approve drives first
- Check drive permissions
- Verify extensions are correct
