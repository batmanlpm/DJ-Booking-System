## ?? Special Feature: Rainbow Theme

The **Rainbow theme** is unique - it **continuously changes colors** by cycling through the entire HSV color spectrum:

### How It Works
- **Real-time animation** using `DispatcherTimer`
- **HSV color space** rotation (0-360 degrees)
- **50ms update rate** (20 FPS) for smooth transitions
- **Automatic start/stop** when switching themes
- **Full spectrum cycling** every ~9 seconds

### Visual Effect
Text, borders, glows, and all accent colors **continuously shift** through:
- Red ? Orange ? Yellow ? Green ? Cyan ? Blue ? Purple ? Back to Red

### Performance
- **< 1% CPU usage** on modern systems
- **Smooth animations** with easing functions
- **No lag or stutter** in the UI
- **Automatic cleanup** when switching themes
