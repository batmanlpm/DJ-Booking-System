## ??? TROUBLESHOOTING

### Issue: "Access Denied"
**Solution**: 
- Close Visual Studio
- Close all applications using project files
- Run PowerShell as Administrator

### Issue: "Path not found"
**Solution**:
- Verify source path exists
- Check spelling of paths
- Ensure no typos in script

### Issue: "Target already exists"
**Solution**:
- Script will ask if you want to delete existing target
- Answer "yes" to proceed
- Or manually delete target directory first

### Issue: Build errors after migration
**Solution**:
```powershell
# Clean and rebuild
dotnet clean
dotnet restore
dotnet build --no-restore
```

### Issue: Visual Studio can't find files
**Solution**:
1. Close Visual Studio
2. Delete `.vs` folder in new location
3. Delete `bin` and `obj` folders
4. Reopen solution
5. Clean and rebuild
