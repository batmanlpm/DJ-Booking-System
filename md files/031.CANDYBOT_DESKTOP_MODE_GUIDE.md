# Candy-Bot Desktop Mode - User Guide

## How to Detach Candy-Bot to Desktop

### Method 1: Right-Click Context Menu
1. **Right-click** on the Candy-Bot avatar (rainbow candy icon in bottom-right corner)
2. Navigate to **"?? Display Options"**
3. Click **"?? Desktop Mode"**
4. Click **"Yes"** when prompted to enable Desktop Widget Mode

### Method 2: Avatar Features
- The floating Candy-Bot avatar appears in the bottom-right corner of the main window
- Right-clicking opens the context menu with full options

## Desktop Widget Features

Once detached, the Candy-Bot Desktop Widget provides:

### ? Core Features
- **Always On Top**: Stays above other windows (toggle in menu)
- **Draggable**: Click and drag anywhere on the screen
- **Resizable**: Drag the edges to resize
- **Transparent Background**: Blends with your desktop

### ?? Quick Actions
- **?? Search**: Instant web search
- **?? Chat**: Open Candy-Bot chat window
- **?? Help**: Access help and documentation

### ?? Personality Modes (6 Available)
1. **?? Normal** - Sweet & Balanced (default)
2. **?? Shy** - Timid & Gentle  
3. **?? Funny** - Jokes & Puns
4. **?? Shit-Stirring** - Sarcastic & Cheeky
5. **?? Raunchy** - Flirty (18+ with age verification)
6. **?? Professional** - Formal & Business-like

### ?? Audio Options
- **?? Quiet Mode**: Only responds when called
- **??? Voice Mode**: Text-to-speech enabled
- **?? Sounds**: Notification sounds and effects

### ?? System Integration
- **?? Start With Windows**: Launch on Windows startup
- **Position Memory**: Remembers last position and size
- **Multi-Monitor Support**: Works across multiple screens

## Closing the Widget

### To Close:
1. Click the **X** button in the title bar
2. Or right-click and select close option

### To Reopen:
- Right-click the main Candy-Bot avatar
- Select "Desktop Mode" again

## Tips & Tricks

?? **Tip 1**: Enable "Start With Windows" to have Candy-Bot ready when you boot up!

?? **Tip 2**: Place the widget in a corner for easy access without blocking your work.

?? **Tip 3**: Try different personalities - each one has unique responses!

?? **Tip 4**: Use Quiet Mode when you need to focus - Candy-Bot won't interrupt.

?? **Tip 5**: The widget saves its position, so it will be where you left it!

## Troubleshooting

### Widget Won't Open
- Restart the main application
- Check if another instance is already running

### Widget Disappears
- It may be off-screen on a disconnected monitor
- Reopen from the main app - it will reset to bottom-right corner

### Personality Not Changing
- Wait a moment after selection
- Look for the glow color change around the avatar
