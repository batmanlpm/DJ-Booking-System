## ?? Common Issues & Fixes

### Issue: Widget won't show
**Fix**: Check if another instance is running
```powershell
Stop-Process -Name "DJBookingSystem" -Force
```

### Issue: Image not displaying
**Fix**: Verify image path in XAML:
```xml
<Image Source="/PNG_Version.png" />
```

### Issue: Build fails (file locked)
**Fix**: Close running instance:
```powershell
Stop-Process -Name "DJBookingSystem" -Force
Start-Sleep -Seconds 2
```
