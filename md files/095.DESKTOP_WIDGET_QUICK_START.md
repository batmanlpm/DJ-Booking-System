## ?? Pro Tips

### Customize Speech Bubble Display Time:
The default is 5 seconds. To change it:
1. Open `CandyBotDesktopWidget.xaml.cs`
2. Find: `Interval = TimeSpan.FromSeconds(5)`
3. Change to your preferred time (e.g., `FromSeconds(10)`)

### Disable Auto-Update Check:
1. Open `CandyBotDesktopWidget.xaml.cs`
2. Find: `await CheckForUpdatesAsync();`
3. Comment it out: `// await CheckForUpdatesAsync();`

### Move Widget to Screen Edge:
- Widget **doesn't snap** to edges yet (planned feature)
- Manually drag to desired position
- Position is **not saved** between sessions (planned)

## ?? Troubleshooting

### Widget Won't Show:
1. Check if already running (look in system tray)
2. Try closing and reopening
3. Check `PNG_Version.png` exists in project root

### Speech Bubble Won't Appear:
- Wait ~1 second after widget loads
- Check XAML for `SpeechBubblePopup` element
- Verify `SpeechBubbleAnimation` storyboard exists

### Context Menu Missing Icons:
- Ensure Unicode emoji support (Windows 10+)
- Check font: `Segoe UI Emoji`
- Update Windows if emoji don't display

### Update Check Fails:
- Check internet connection
- Update server might be down (safe to ignore)
- Widget still works without updates
