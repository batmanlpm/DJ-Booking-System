## üé® IMAGE GENERATION CHEAT SHEET

### Initialize
```csharp
var imageGen = new CandyBotImageGenerator(
    apiKey: "sk-your-key",
    provider: CandyBotImageGenerator.ImageProvider.OpenAI_DALLE3,
    soundManager: _candySound
);
```

### Generate Image
```csharp
var result = await imageGen.GenerateImageAsync(
    "cyberpunk DJ booth with neon green lights",
    size: ImageSize.Square_1024,
    quality: ImageQuality.HD
);

if (result.Success)
    Console.WriteLine($"Saved: {result.LocalFilePath}");
```

### DJ Presets (Quick)
```csharp
await imageGen.GenerateDJImageAsync(DJImagePreset.NeonClubScene);
await imageGen.GenerateDJImageAsync(DJImagePreset.VinylCollection);
await imageGen.GenerateDJImageAsync(DJImagePreset.LivePerformance);
await imageGen.GenerateDJImageAsync(DJImagePreset.StudioSetup);
await imageGen.GenerateDJImageAsync(DJImagePreset.EventPoster);
```

### Batch Generate
```csharp
var prompts = new[] { "prompt1", "prompt2", "prompt3" };
var batch = await imageGen.GenerateImageBatchAsync(prompts);
// batch.SuccessfulImages, batch.FailedImages
```

## üìÑ DOCUMENT GENERATION CHEAT SHEET

### Initialize
```csharp
var docGen = new CandyBotDocumentGenerator(_candySound);
```

### Quick Documents
```csharp
// Text file
await docGen.CreateTextFileAsync("content");

// Word
await docGen.CreateWordDocumentAsync("title", "content");

// PDF
await docGen.CreatePDFDocumentAsync("title", "content");

// Markdown
await docGen.CreateMarkdownFileAsync("title", "content");

// Open file
docGen.OpenDocument(result.FilePath);
```

### DJ Setlist
```csharp
var tracks = new List<Track>
{
    new Track { Artist = "Artist", Title = "Song", Duration = "4:30" }
};

await docGen.CreateDJSetlistAsync(
    "DJ Name", "Venue", DateTime.Now, tracks
);
```

### Booking Contract
```csharp
await docGen.CreateBookingContractAsync(
    djName: "DJ CANDY",
    venueName: "The Club",
    eventDate: DateTime.Parse("2024-12-31"),
    fee: 500.00m,
    additionalTerms: "Equipment provided"
);
```

### Event Flyer
```csharp
await docGen.CreateDJFlyerAsync(
    "DJ Name", "Event Name", DateTime.Now,
    "Venue", "Extra info"
);
```

## üêõ QUICK TROUBLESHOOTING

**Image Gen - "Rate limit"**
‚Üí Wait 60 seconds or upgrade plan

**Both - "File not found"**

- Images: `%USERPROFILE%\Pictures\CandyBot-Generated`
- Docs: `%USERPROFILE%\Documents\CandyBot-Documents`
