## üéØ How to Use

### Voice Playback

```csharp
// Initialize
var soundManager = new CandyBotSoundManager();
soundManager.SetVoiceMode(true);
soundManager.SetSoundsEnabled(true);

// Play specific line
await soundManager.PlayVoiceLine("001");

// Play random from category
await soundManager.PlayRandomVoiceFromCategory("Greeting");

// Context methods
soundManager.PlayGreeting();
soundManager.PlayBookingResponse();
soundManager.PlayPositiveFeedback();
```

### File Management

```csharp
// Initialize
var fileManager = new CandyBotFileManager();

// Request permission (required)
fileManager.AddApprovedFolder(@"C:\Users\GOD\Music");

// Search files
var musicFiles = await fileManager.FindMusicFilesAsync();
var setlists = await fileManager.FindDJSetlistsAsync();

// Organize
int organized = await fileManager.OrganizeFilesByTypeAsync(@"C:\Downloads");

// Get stats
var stats = await fileManager.GetFileStatisticsAsync();
```
