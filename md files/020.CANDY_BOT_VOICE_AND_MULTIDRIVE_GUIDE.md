## üîß Configuration Options

### Adjust Candy's Voice

In `CandyBotWindow.xaml.cs` constructor, modify:

```csharp
_tts.SetSpeechRate(1);  // -10 to 10 (0 = normal, higher = faster)
_tts.SetVolume(80);     // 0 to 100
```

### Change Available Voices

```csharp
// Get all available voices
var voices = _tts.GetAvailableVoices();

// Select a specific voice
_tts.SelectVoice("Microsoft Zira Desktop");  // or any installed voice
```

### Customize Search Exclusions

In `MultiDriveFileSearcher.cs`, modify the default excluded folders:

```csharp
excludeFolders = new List<string>
{
    "Windows", "System32", "$Recycle.Bin",
    "ProgramData", "Program Files", "Program Files (x86)",
    "AppData", "node_modules", ".git"  // Add your own
};
```

## üêõ Troubleshooting

### "Voice not working"
- Ensure Windows Speech Synthesis is installed
- Check that `_voiceEnabled = true` in code
- Try different Windows voices in Settings > Time & Language > Speech

### "Cannot access drive X"
- Admin rights may be needed for system drives
- Some drives might be disconnected/unmapped
- Click "Refresh Drives" to reload

### "Search too slow"
- Reduce number of drives selected
- Use specific path instead of full drive
- Add more excluded folders
