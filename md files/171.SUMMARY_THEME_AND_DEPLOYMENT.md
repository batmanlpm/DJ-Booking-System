## ?? Special Feature: Rainbow Theme

### What Makes It Special:
- **Continuously animated** - Colors cycle through entire spectrum
- **HSV color space** - Smooth, natural color transitions
- **20 FPS animation** - Updates every 50ms
- **Auto start/stop** - Automatically manages animation lifecycle
- **Performance optimized** - < 1% CPU usage
- **Full spectrum** - Red ? Orange ? Yellow ? Green ? Cyan ? Blue ? Purple ? repeat

### How It Works:
```
1. User selects "Rainbow" theme
2. RainbowThemeAnimator.StartAnimation() called automatically
3. DispatcherTimer updates every 50ms
4. HSV hue rotates 0-360 degrees
5. AccentColor and AccentBrush update dynamically
6. All UI elements reflect current rainbow color
7. Animation stops when switching to different theme
```
